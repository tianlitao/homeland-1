Homeland::Wiki
--------------

Wiki 栏目插件 for [Homeland](https://gethomeland.com)

简单的实现 /wiki 栏目，并在导航栏显示。

## Usage

## Installation

[Download ZIP](https://github.com/homeland-plugins/wiki/archive/master.zip) and visit Homeland / Admin / Plugins to install.
